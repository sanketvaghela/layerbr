# layerbr 🧅

Like keybr, but for torturing yourself with the OSI and TCP/IP models.

Are you studying for your CCNA, CompTIA Network+, or just trying to survive a whiteboard interview with a grumpy sysadmin? Do you find yourself muttering "Please Do Not Throw Sausage Pizza Away" in your sleep, only to wake up in a cold sweat because you forgot which layer IPsec lives on?

Welcome to layerbr. It's a rapid-fire, muscle-memory trainer that drills network layers into your brain until you can map "Data Link" to "Layer 2" faster than a misconfigured router can drop a packet.

## ✨ Features

- **Keyboard First:** Mash 1, 2, 3, or 4. Touch your mouse and you should feel a mild sense of shame (just kidding, mobile touch works perfectly too).
- **Ruthless Stats:** Watch your accuracy plummet and your "Slowest Time" mock you in real-time.
- **Dark Mode:** Because we are developers, not flashbang test subjects.
- **Zero Dependencies:** No `npm install`. No `node_modules` folder heavier than a dying star. Just pure, unadulterated HTML/JS/CSS goodness.

## 🚀 How to Play

1. Clone this repo (or literally just double-click the file).
2. Open `index.html` in any browser.
3. Start sweating.

## 🛠️ Built With

- **HTML & Vanilla JavaScript:** Like it's 1999, but with arrow functions.
- **Tailwind CSS:** Dragged in via CDN because setting up Webpack/PostCSS for a single file is a crime against humanity.
